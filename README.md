# ZAD Website Redesign v13

This version keeps the v12 direction but removes the cramped insurance workflow demo from the homepage.

## Structure
- `index.html` — English homepage
- `services.html` — English Solutions page
- `cases.html` — English Cases page with a full insurance operations workflow case
- `about.html` — English Company page
- `contact.html` — English Contact page
- `/fr/` — French version of the same structure
- `styles.css` — shared styles
- `script.js` — menu, tabs, ROI estimator, contact intake, and interaction logic

## Main v13 changes
- Adds a dedicated Cases page.
- Replaces the unreadable homepage workflow mini-demo with a clean case teaser.
- Builds a full-width insurance workflow workspace with readable panels:
  - intake workspace
  - agent assistant
  - new-hire coach
  - manager cockpit
- Adds Cases to the navigation and footer in English and French.

Open `index.html` locally in a browser to test.
